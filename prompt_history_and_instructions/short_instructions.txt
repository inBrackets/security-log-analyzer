How to run:
1. Execute the main.py file in the root directory
2. Follow the instructions in the interaction mode. Sample flow:
* Path [logs]:
* 1) narrative  attack story grouped by IP (default)
* 5) INFO  everything
* Time range filter: (leave blank and press enter twice)

Run with generated data:
1. Execute the generate_live_logs.py file
2. Find the new log files in directory /logs-live
3. Run the main.py, but with the different directory in the interactive mode
* Path [logs]: logs-live
* 1) narrative  attack story grouped by IP (default)
* 5) INFO  everything
* Time range filter: (leave blank and press enter twice)
